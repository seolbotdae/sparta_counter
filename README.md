# 카운터 앱
## 기능

1. 숫자 증가/감소:
  - "+" 버튼을 클릭하면 숫자가 1씩 증가.
  - "-" 버튼을 클릭하면 숫자가 1씩 감소.
2. 초기화 기능:
  - "Reset" 버튼을 클릭하면 숫자가 0으로 초기화.
3. 현재 숫자 표시:
  - 중앙에 UILabel로 현재 숫자를 표시.
4. 0 에서 10 까지의 범위로 카운트 제한
  - 최소 값일때 "-" 버튼 비활성화, 최대 값일때 "+" 버튼 비활성화
5. Auto Layout으로 UI 배치하여 다양한 화면 크기에서 정상적으로 표시됨
